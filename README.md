# 磨耳朵 · Podcast Reader

把文字变成播客，像平时听播客一样磨耳朵。

## 使用方法

1. 用浏览器打开 `index.html`
2. 粘贴文字或拖入 `.txt` `.md` 文件
3. 点击播放按钮或按空格键开始朗读
4. 可以调速、换声音、快进快退
5. 点击「保存」可保存为播客集，下次继续听

## 快捷键

| 按键 | 功能 |
|---|---|
| 空格 | 播放 / 暂停 |
| ← → | 快退 / 快进 15 秒 |
| Esc | 停止播放 |

## 移动端

在手机上用浏览器打开同样可用，响应式布局自动适配。

## 技术

纯前端，基于浏览器 Web Speech API，无需安装任何依赖。
